SELECTORES SIMPLES

Selector de etiqueta

Se utiliza para seleccionar y aplicar estilos a todos los elementos de una etiqueta específica en una página web.

Selector de clase

Se utiliza para seleccionar y aplicar estilos a elementos que tienen un atributo 'class' específico.

Selector de ID

Se utiliza para seleccionar y aplicar estilos a un elemento específico que tiene un atributo 'id' único.

Selector universal

Selecciona todos los elementos de una página.
